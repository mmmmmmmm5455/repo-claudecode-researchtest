## Bugs Found

### BUG-20260430-001 — Shader: uTimeWobble Undeclared
- **Severity:** HIGH | **Category:** rendering | **Auto-fixable:** Yes
- PS1 vertex shader uses `uTimeWobble` uniform without declaring it. Affects MeshStandardMaterial, MeshBasicMaterial, PointsMaterial — all 3D rendering broken.
- Fix: Add `uniform float uTimeWobble;` to `js/shaders/ps1_vertex.glsl`

### BUG-20260430-002 — Emotion/Infection Always Default
- **Severity:** HIGH | **Category:** stub | **Auto-fixable:** No
- `Player` class has no `emotion`/`infection` attributes. `getattr(state.player, 'emotion', 50.0)` always returns default. All game mechanics dependent on emotion/infection variation are non-functional.

### BUG-20260430-003 — ThinkingBlock LLM Error
- **Severity:** MEDIUM | **Category:** crash | **Auto-fixable:** No
- Intermittent. DeepSeek v4 Pro returns `ThinkingBlock` objects client can't handle. Graceful fallback to error narrative.

### BUG-20260430-004 — Are-You-Real Popup Never Appears
- **Severity:** HIGH | **Category:** UI | **Auto-fixable:** Yes
- `GameRenderer.showSystemDialog('are_you_real')` called but `#are-you-real-popup` retains `class="hidden"`. Popup never becomes visible.
